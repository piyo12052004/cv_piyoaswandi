## 🚀 Fitur Utama
- **Responsive Design:** Dioptimalkan untuk perangkat mobile, tablet, hingga desktop menggunakan utilitas kelas Tailwind CSS.
- **Modern Typography & Layout:** Menggunakan layout bergaya minimalis yang berfokus pada keterbacaan data (*scannability*).
- **Print-Friendly:** Menggunakan rasio kontras tinggi yang aman dan rapi jika rekruter ingin mencetaknya atau menyimpannya sebagai PDF.

## 🛠️ Teknologi yang Digunakan
- **Frontend:** HTML5, Tailwind CSS (via CDN), FontAwesome v6 (untuk ikon).
- **Deployment & Hosting:** GitHub Pages.
